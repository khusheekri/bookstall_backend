Bookstall-Backend Docs

This is an Bookstall application backend server which has a few endpoints, written in Typescript using node.js and express.js

  • Path: /signup Description: Signup and create a new user in the database, returns a token
  • Path: /login Description: Login to the system with the email and password
  • Path: /api/books Description: Get request gets all the book in json, Post request adds books to the Database
